Privacy

Last updated: June 2, 2026

This Privacy Policy describes how LIFEHUB handles information, user progress, and productivity data while using the platform.

LIFEHUB was designed with privacy as a core principle. Most user information remains stored locally within the user's browser and is used solely to improve the user experience.

Data Collection

LIFEHUB does not require users to submit personal information in order to access core productivity features.

Tasks, habits, achievements, and productivity statistics are primarily managed within the user's local environment.

Local Storage

To maintain progress between sessions, LIFEHUB may store information using browser local storage.

This data remains on the user's device and is not transmitted to external servers.

User Rights

Users maintain full control over their locally stored data.

Information can be removed at any time by clearing browser storage.
